# Todo App

React + Vite ile yaptığım basit bir yapılacaklar listesi uygulaması. Arayüz Tailwind CSS ile stillendirildi.

## Özellikler

- Görev ekleme, silme ve tamamlandı olarak işaretleme
- Tümü / Aktif / Bitti şeklinde filtreleme
- İlerleme çubuğu (kaç görev tamamlandı)
- Tamamlanan görevleri tek seferde temizleme
- Veriler localStorage'da tutuluyor, sayfa yenilense de kaybolmuyor

## Kullanılan Teknolojiler

- React 19
- Vite
- Tailwind CSS v4

## Kurulum

```bash
npm install
npm run dev
```
